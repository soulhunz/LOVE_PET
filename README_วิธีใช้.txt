🐾 LOVE_PET — วิธีติดตั้ง / รัน / อัพเดท
=========================================

ครั้งแรก (ติดตั้ง)
------------------
  ดับเบิลคลิก  install.bat
  → ตรวจ Python ให้, ตั้งเปิดอัตโนมัติตอนเปิดเครื่อง,
    สร้างไอคอนบนเดสก์ท็อป, และเปิดตัวละครให้เลย

เปิดเอง
-------
  • เปิดเองทุกครั้งที่เปิดเครื่อง (ออโต้รัน)
  • หรือดับเบิลคลิกไอคอน LOVE_PET บนเดสก์ท็อป
  • หรือดับเบิลคลิก  start_pet.vbs

อัพเดทโค้ดใหม่ (มี 2 วิธี)
--------------------------
  1) คลิกขวาที่ตัวละคร → "🔄 อัพเดทโปรแกรม"
  2) ดับเบิลคลิก  update.bat
  → ดึงโค้ดล่าสุดจาก GitHub แล้วเปิดใหม่ให้อัตโนมัติ
    (เซฟเกม save.json จะถูกเก็บไว้ ไม่หาย)

ปิดการเปิดอัตโนมัติ
-------------------
  ดับเบิลคลิก  uninstall.bat
  (ไฟล์โปรแกรมและเซฟเกมยังอยู่ครบ)

สิ่งที่ต้องมี
------------
  • Python 3  (https://www.python.org/downloads/  ติ๊ก Add to PATH)
  • Git       (https://git-scm.com/download/win   เฉพาะตอนใช้ update)

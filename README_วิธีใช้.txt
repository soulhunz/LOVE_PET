🐾 LOVE_PET — วิธีติดตั้ง / รัน / อัพเดท
=========================================

โครงสร้าง 2 เวอร์ชัน (git branch)
---------------------------------
  • main = เวอร์ชัน SERVER (เสถียร, รันจริง, ออโต้สตาร์ท, ปุ่มอัพเดทดึงอันนี้)
  • dev  = เวอร์ชัน DEV (ไว้แก้/ทดลอง)

  ปกติให้โฟลเดอร์อยู่ที่ branch main (เวอร์ชันเสถียร)
  จะแก้โค้ดค่อยสลับไป dev ด้วย dev_mode.bat


ครั้งแรก (ติดตั้ง)
------------------
  ดับเบิลคลิก  install.bat
  → ตรวจ Python, ตั้งเปิดอัตโนมัติตอนเปิดเครื่อง, สร้างไอคอนเดสก์ท็อป, เปิดให้เลย


เปิดโปรแกรม
-----------
  • เปิดเองตอนเปิดเครื่อง / ไอคอนเดสก์ท็อป / start_pet.vbs
  • ปุ่ม ⚙ บนแถบ = ตั้งค่า (อัพเดท / เกี่ยวกับ / ออก)


วงจรการทำงาน (สำคัญ)
--------------------
  1) อยากแก้โค้ด        →  dev_mode.bat   (สลับไป branch dev)
  2) แก้เสร็จ/อยากเซฟ   →  publish.bat    (commit + push branch ปัจจุบันขึ้น GitHub)
  3) พร้อมปล่อยเสถียร   →  release.bat    (รวม dev เข้า main แล้ว push main)
  4) เครื่องที่รันจริง   →  update.bat หรือปุ่ม ⚙→อัพเดท
                            (สลับไป main + ดึงเวอร์ชัน server ล่าสุด)

  ทุกสคริปต์จะ "เช็คก่อนสลับ branch" — ถ้ามีงานยังไม่เซฟจะเตือนและหยุด
  ไม่ลบงานทิ้งเด็ดขาด (ต่างจากของเดิมที่ใช้ reset --hard)


ปิดการเปิดอัตโนมัติ
-------------------
  ดับเบิลคลิก  uninstall.bat   (ไฟล์/เซฟยังอยู่ครบ)


ถ้าโปรแกรมเปิดไม่ขึ้น
---------------------
  ดับเบิลคลิก  run_debug.bat   (เห็น error ตรง ๆ) แล้วดูไฟล์ error.log


สิ่งที่ต้องมี
------------
  • Python 3  (https://www.python.org/downloads/  ติ๊ก Add to PATH)
  • Git       (https://git-scm.com/download/win)
  • ล็อกอิน GitHub ที่มีสิทธิ์เขียน repo (push ครั้งแรกจะให้ล็อกอิน)
